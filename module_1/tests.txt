Тестовый сценарий 1.1.1. Регистрация нового пользователя
Предусловия:
Пользователь находится на главной странице сайта, пользователь является новым и ранее не регистрировался на данном сайте
Шаги:
- Шаг 1. Польователь нажимает кнопку <<Войти или зарегестрироваться>> в верхнем правом углу на сайте.
- - Проверка: Открыты две формы. В левой стороне форма для входа, с правой сторноны форма для регистрации.
- - - Ожидаемый результат: Открыты формы для регистрации и для входа
- Шаг 2. Пользователь вводит валидные данные в поля <<Адрес электронной почты>>, <<Пароль>>, <<Повторите пароль>>
- - Проверка: Открыта главная страница сайта с благодарность за регистрацию, пользователь авторизован
- - - Ожидаемый результат: Открыта страница с текстом "Cпасибо за регистрацию", полученно электронное письмо для 
      потверждения адреса эл. почты.

Тестовый сценарий 1.2.1 Вход на сайт
Предусловия:
Пользователь находится на странице входа и регистрации, пользователь заресгистрирован на данном сайте.
Шаги:
- Шаг 1. Пользователь вводит валидные данные в  поля <<Адрес электронной почты>>, <<Пароль>>.
- - Проверка: Открыта главная страница сайта с текстом <<Рады видеть вас снова>>, пользователь авторизован.
- - - Ожидаемый результат: Открыта страница с даннимы пользователя, либо открыта страница с текстом "Вы успешно
       авторизованы"

Тестовый сценарий 2.1.1. Добавление одного товара в корзину
Предусловия:
Пользователь авторизован, находится на гланой странице.
Шаги:
- Шаг 1. Пользователь нажимает в левой стороне на вкладку все товары.
- - Проверка: Открыта страница с перечнем товаров, под товарами стоят отметки о наличии их на скдаде.
              В верху указано место нахождения пользователя - <<Начало/Все товары>>.
- - - Ожидаемый результат: Открывается страница с перечнем всех товаров.
- Шаг 2. Пользователь выбирает первый товар - книгу <<The shellcoder's handbook>> под товаром стоит галочка
         <<На складе>> и нажимает на кнопку <<Добавить в корзину>>
- - Проверка: Пользоваель остался на странице выбора товаров, появился текст <<The shellcoder's handbook
    был добавлен в вашу корзину>>, <<Ваша корзина удовлетворяет условиям предложения Shipping offer>>, <<Стоимость
    корзины теперь составляет 9,99 £ >>, появились кнопки <<Посмотреть корзину>> и <<Оформить>>
- - - Ожидаемый результат: Пользователь остаётся на странице выбора товаров с появлением текста об успешном добавление
      товара и с повлением кнопок для оформление заказа и перехода в корзину.

Тестовый сценарий 2.2.1. Поиск товара с помощью вспомогательного меню на сайте.
Предусловия:
Пользователь является зарегистрированным и авторизованным на сайте, пользователь вводит название имеющегося товара
на сайте и наличием на складе. Название книги - <<The shellcoder's handbook>>.
Шаги:
- Шаг 1. Пользователь вводит название товара для поиска книга - << The shellcoder's handbook >> в верхее правое меню
         сайта, преднозначенное и нажимает на кнопку поиска.
- - Проверка: Открыта страница с найденным одним товаром. С книгой <<The shellcoder's handbook>> с кнопкой добавить
              данный товар в корзину.
- - - Ожидаемый результат:  Открыта страница с найденным товаром, наличие кнопок с добавлением найденого товара
                            в корзину. Предложения для покупки похожего товара.

Тестовый сценарий 2.2.2 Обзор страницы товара, найденного с помощью вспомогательного меню на сайте.
Предусловия:
Пользователь является зарегистрированным и авторизованным на сайте,  пользователь находится на странице с товаром 
книга - <<The shellcoder's handbook>>. Товар найден с помощью вспомогательного меню для поиска на сайте. Товар есть 
в наличии у продавца на сайте.
- Шаг 1. Пользователь нажимает на название товара.
- - Проверка: Открыта страница с описанием товара и его характирестиками, есть кнопка добавления отзыва об данном товаре
              и прочитать отзывы об товаре. На странице имеются кнопки добавления товара в корзину и добавить товар в
              список желаемого, присутсвет информация о стоимости товара и его наличии на складе продавца.
- - - Ожидаемый результат: Открыта страница с описанием товара, его рейтнгом и с возможностью прочитать отзывы о
                           товаре. Наличие копки для добавления товара в корзину.
